# YoChick!

Game edukasi tentang merawat ayam yang dibuat menggunakan godot engine.

## Installation

Download [YoChick!](https://github.com/daffawrdhn/YoChick-) dan masukan kedalam folder project.

```bash
Create new Project -> New Folder Project -> Scan
```
